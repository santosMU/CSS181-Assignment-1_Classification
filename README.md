# assignment 1 – setup guide (for collaborators)

---

## prerequisites

### 1. install Python
https://www.python.org/downloads/

### 2. if you're using vscode
install extensions:
- Python
- Jupyter

### 3. go to terminal and run
```
pip install pandas numpy scikit-learn matplotlib seaborn
```

### 4. git workflow
when you made a change, go to terminal and:
```
git add .
git commit -m "Describe your change"
git push
```
